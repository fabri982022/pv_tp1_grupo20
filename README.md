# Gestión de Proyectos Educativos - TP1 Grupo 20

## Descripción del Proyecto

Plataforma educativa colaborativa para la gestión y seguimiento de proyectos académicos. Este proyecto implementa una interfaz web intuitiva que permite a estudiantes y docentes crear, organizar y gestionar proyectos académicos de forma eficiente.

## Páginas Implementadas

### 1. **index.html** - Dashboard Principal
- Página de inicio con navegación principal
- Sección de estadísticas del sistema:
  - Proyectos totales
  - Tareas para hoy
  - Mensajes pendientes
  - Equipo activo
- Sección de "Últimas Novedades" con actividades recientes del equipo
- Diseño responsive con estructura semántica

### 2. **proyectos.html** - Explorador de Proyectos
- Listado completo de proyectos disponibles
- Sistema de filtros avanzados:
  - Filtro por categoría (Web, Móvil, Datos, Sistemas)
  - Filtro por año
  - Filtro por estado (En Curso / Finalizado)
- Visualización de proyectos en tarjetas con:
  - Imagen representativa
  - Nombre y descripción
  - Categoría y estado
  - Enlace a detalle del proyecto

### 3. **detalle.html** - Página de Detalle del Proyecto
- Información completa del proyecto seleccionado
- Secciones incluidas:
  - Encabezado con datos generales y estado
  - Descripción extendida del proyecto
  - Recursos y enlaces relacionados (especificaciones, prototipos, repositorio, documentación)
  - Equipo del proyecto con roles y responsabilidades
  - Miembros con información detallada de cada integrante

### 4. **perfil.html** - Perfil de Usuario
- Página de perfil personal del usuario
- Secciones implementadas:
  - Información personal (foto, nombre, rol, especialidad, institución, contacto)
  - Resumen profesional
  - Proyectos en progreso
  - Listado de habilidades técnicas
  - Historial de actividad reciente
  - Acciones disponibles (editar perfil, cambiar contraseña, descargar CV)

## Características Principales

-  Navegación coherente entre todas las páginas
-  Estructura HTML semántica y accesible
-  Diseño responsive con viewport meta tag
-  Sistema de filtros interactivo
-  Contenido dinámico y organizado
-  Información estructurada de proyectos y usuarios
-  Interfaz amigable para la gestión académica

## Tecnologías Utilizadas

- HTML5
- Estructura semántica
- Meta tags para responsive design
- Navegación con listas y enlaces

## Notas de Implementación

- Todas las páginas comparten una estructura de navegación consistente
- Los enlaces internos permiten navegar fluidamente entre páginas
- El contenido está centrado en la gestión educativa y académica
- Se utilizan clases CSS descriptivas para facilitar el estilizado posterior

## Instrucciones de Uso

1. Abrir `index.html` en un navegador web para acceder a la plataforma
2. Utilizar los enlaces de navegación para moverse entre secciones
3. Explorar los proyectos desde la página "Proyectos"
4. Ver detalles específicos de cada proyecto
5. Acceder al perfil del usuario desde la navegación principal
