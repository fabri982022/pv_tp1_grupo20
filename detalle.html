App Móvil para Gestión de Tareas

Fecha de Inicio: 15 de Marzo de 2026

Autores: María González, Juan Rodríguez, Carlos López

Estado: En Curso

Descripción del Proyecto

Este proyecto tiene como objetivo desarrollar una aplicación móvil multiplataforma que permita a los usuarios crear, organizar y gestionar sus tareas diarias de forma eficiente. La aplicación incluirá funcionalidades avanzadas como recordatorios automáticos, categorización de tareas, establecimiento de prioridades y sincronización en tiempo real entre dispositivos.

La aplicación será desarrollada utilizando tecnologías modernas como React Native para garantizar compatibilidad con plataformas iOS y Android. Se implementará un sistema de autenticación seguro, almacenamiento en la nube y una interfaz intuitiva que proporcione una experiencia de usuario óptima. El proyecto busca mejorar la productividad de estudiantes y profesionales mediante herramientas simples pero poderosas.
